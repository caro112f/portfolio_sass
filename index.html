    Hacked Hogwarts

    The Task:

    To show a list of students and their info via JSON and API, where the user can filter, sort, remove/expel, and give selected students unique statuses. The user can also find a sneaky hacking button, that causes different student information to be randomized as well as inserting a new non-student into the array.

    How I solved it:

    By using a lot of JavaScript functions with both fetch, return values, parameters and closures. I also visualized all of the code using detailed and generalised Activity Diagrams. Styling and responsiveness was ensured with CSS, where I used Grid, Flex and Media Queries.

    My favourite part:

    Coding the hacking function, because of the way it could manipulate all of the previous code. Seeing all the functions come together to make the site very interactive and fun. Styling the page so the information inside the student list was user-friendly, readable and theme-appropriate (Harry Potter Universe).

    Stay Healthy Game

    The Task:

    In this first semester assignment I had to design and develop a simple game with UI-components and characters that followed a design genre. This was my first assignment using JavaScript and CSS-animations.

    How I solved it:

    The design aspect was achieved through research on different design genres for game characters, a design sprint to create a general idea of the style and making the backgrounds and characters "come to life" using Adobe Illustrator. I used both Activity Diagrams and State Machine Diagrams to plan the code and functionality of the game and went to work to make the game interactive and inspiring with a lot of CSS and Javascript.

    My favourite part:

    I loved being creative with the story behind the game and making my characters cute and believable. Getting to know how useful and powerful JavaScript is in combination with CSS was eye-opening and fun.
